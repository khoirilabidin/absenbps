Halaman Administrator pada aplikasi Absensi dan Laporan Kegiatan Magang berfungsi untuk mengelola akun pengguna yang memiliki hak akses sebagai
admin. Melalui halaman ini, admin utama dapat menambahkan, mengubah, dan menghapus data administrator yang dapat mengakses sistem. 
Fitur ini digunakan untuk mengatur hak akses pengelola sistem sehingga pengelolaan data absensi, data mahasiswa, serta kegiatan magang dapat
dilakukan secara terkontrol dan terorganisir.
